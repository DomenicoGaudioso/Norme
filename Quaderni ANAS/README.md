# Quaderni Tecnici ANAS

> 🚧 **Cartella riservata** ai Quaderni tecnici ANAS per ponti e infrastrutture.

## Documenti attesi

I Quaderni ANAS sono documenti tecnici di riferimento per la progettazione, costruzione e manutenzione delle infrastrutture stradali e dei ponti:

- **Quaderno ANAS n.1** — Linee guida per il progetto di ponti stradali.
- **Quaderno ANAS n.2** — Linee guida per il progetto delle gallerie artificiali.
- **Quaderno ANAS n.3** — Linee guida per il progetto delle opere in terra.
- **Quaderno ANAS n.4** — Linee guida per la manutenzione dei ponti.
- **Quaderni ANAS sulle barriere paramassis** (progettazione, esecuzione, manutenzione).
- **Capitolato ANAS speciale d'appalto per l'esecuzione di ponti.**
- **Capitolato ANAS speciale d'appalto per le opere in c.a. e c.a.p.**
- **Linee guida ANAS per le ispezioni sui ponti** (Scheda SVP, Scheda SDO).

## Stato

Nessun documento presente in questa cartella.

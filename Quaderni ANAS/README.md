# Quaderni Tecnici ANAS

I **Quaderni Tecnici ANAS** sono documenti di riferimento per la progettazione, l'esecuzione e la manutenzione delle infrastrutture stradali, dei ponti e delle gallerie.

Pubblicati da **ANAS SpA**, sono **gratuiti** per consultazione personale ma coperti da copyright. **Non sono redistribuiti in questa cartella**: per ogni documento è fornito il link diretto al sito ufficiale ANAS.

Pagina indice ufficiale ANAS: **<https://www.stradeanas.it/it/documenti-tecnici>**

## ⚠️ Documenti NON presenti come Quaderni ANAS autonomi

Per chiarezza, alcuni argomenti comunemente richiesti **non hanno un Quaderno ANAS dedicato**:

| Voce | Documento reale di riferimento |
|------|--------------------------------|
| Progettazione ponti nuovi | NTC 2018 + Eurocodici + capitolati ANAS (per gara) |
| Linee guida ispezioni ponti (SVP, SDO) | **DM 493/2021** del MIT/CSLP (vedi cartella `../Linee Guida Ponti/`) |
| Gestione patrimonio ponti | **DM 493/2021** del MIT/CSLP |
| Capitolato speciale d'appalto ponti / c.a. / c.a.p. | Allegato a ogni singolo bando di gara ANAS (non esiste PDF statico pubblico) |
| Barriere paramassi | Riferimento: **UNI 11211-4** (non c'è Quaderno ANAS dedicato) |
| Linee guida gallerie stradali esistenti | **DM 247/2023** del MIT/CSLP |

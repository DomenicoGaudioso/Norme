Documenti Tecnici del **Consiglio Nazionale delle Ricerche** (Commissione di studio per la predisposizione e l'analisi di norme tecniche relative alle costruzioni).

I documenti CNR-DT sono **gratuiti per consultazione personale** ma hanno licenza "Proprietà letteraria riservata CNR": **non sono redistribuiti in questa cartella**. Per ogni documento è fornito il link diretto alla **pagina ufficiale CNR**, dove scaricare il PDF per uso personale.

Pagina indice ufficiale CNR: **<https://www.cnr.it/it/norme-tecniche-costruzioni>**

## 🔗 Come scaricare i PDF

Per ogni documento il PDF ufficiale è disponibile alla pagina indice CNR:
> **<https://www.cnr.it/it/norme-tecniche-costruzioni>**

Pattern URL dei PDF diretti (verificati sul dominio CNR):

```
https://www.cnr.it/sites/default/files/public/media/norme_tecniche_costruzioni/IstruzioniCNR_DT<NNN>_<AAAA>.pdf
```

Esempi:
- DT 207/2008 (vento): `IstruzioniCNR_DT207_2008.pdf`
- DT 206 R1/2014 (legno): `IstruzioniCNR_DT206_R1_2014.pdf`
- DT 210/2013 (vetro): `IstruzioniCNR_DT210_2013.pdf`
- DT 213/2015 (ponti muratura): `IstruzioniCNR_DT213_2015.pdf`

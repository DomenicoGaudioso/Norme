# CNR — Documenti Tecnici del Consiglio Nazionale delle Ricerche

> 🚧 **In preparazione** — cartella da popolare.

## Documenti previsti

Serie **CNR-DT** (Documenti Tecnici):
- **CNR-DT 200/2011** — Istruzioni per la progettazione, l'esecuzione ed il controllo di strutture di rinforzo con materiali compositi FRP.
- **CNR-DT 200 R1/2013** — Revisione del DT 200/2011 (FRP).
- **CNR-DT 206/2007** — Istruzioni per la progettazione, l'esecuzione ed il controllo di strutture di legno.
- **CNR-DT 207/2008** — Istruzioni per la valutazione della sicurezza sismica di edifici esistenti (prima della Circolare 2019).
- **CNR-DT 207 R1/2014** — Istruzioni per la progettazione, esecuzione e controllo di interventi di rinforzo con materiali compositi FRCM.
- **CNR-DT 212/2013** — Istruzioni per la progettazione, esecuzione e controllo di interventi di consolidamento statico mediante l'utilizzo di compositi fibrorinforzati.
- **CNR-DT 214/2018** — Linee guida per la valutazione e la riduzione del rischio sismico di edifici esistenti in c.a.
- **CNR-DT 215/2018** — Istruzioni per la progettazione, l'esecuzione ed il controllo di statically non-reinforced masonry.
